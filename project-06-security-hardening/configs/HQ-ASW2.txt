enable
configure terminal

! --- Neighbor Discovery ---
! WHY: LLDP gives a vendor-neutral topology check alongside CDP.
lldp run

! --- ATTACKER1 Access Port Security ---
! WHY: This port is the controlled rogue-device test point for Project 06.
interface Ethernet1/1
 description ACCESS-ATTACKER1-VLAN100-SECURITY-TEST
 switchport mode access
 switchport access vlan 100
 switchport nonegotiate
 spanning-tree portfast
 spanning-tree bpduguard enable
 switchport port-security
 switchport port-security maximum 2
 switchport port-security mac-address sticky
 switchport port-security violation restrict
 no shutdown

end
write memory
